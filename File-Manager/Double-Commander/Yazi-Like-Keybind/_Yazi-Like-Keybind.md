# Yazi-Like-Keybind.md

## Notes

* Yazi uses mostly single-alphabet, modifier keys like `Ctrl`, `Alt`, `Shift`, `Windows-Key`, `Tab` for it's keybindings.
  * It can be set on double-commander's `shortcuts.scf` file.
  * Double-commander does not support double-alphabet like keybindings like `yy`, `dd` etc.
  * No need to use AutoHotkey software. It is only required, when using double-alphabet like keybindings like `yy`, `dd` etc.

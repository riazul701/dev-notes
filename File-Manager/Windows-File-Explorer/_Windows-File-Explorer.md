# Windows-File-Explorer.md

## Notes

* Use `SSHFS` to mount HDD from `SSH` connection. Windows-File-Explorer does not natively support `SFTP` like PCManFM, Thunar (Linux) file manager address bar syntax `sftp://{{ip-address}}`

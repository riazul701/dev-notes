* Build starter kit for both Learning and CodeCanyon
* This starter kit will include Authentication, User Management, Email Verification, User Role Management, Two Factor Authentication, CRUD, Report etc.
* For CRUD (Create, Read, Update, Delete) use inventory software system.
* Make different versions of this starter kit -- Laravel and Bootstrap, Laravel and VueJS, Laravel and ReactJS, Laravel and Angular JS
* Also build starter kit for -- React Native, Cordova and OnsenUI
* Use this starter kit for starting point of your own project.
* Make Automatic CRUD (Create, Read, Update, Delete) generator using html forms. This form will create files in project. Make CRUD generator as a Composer package and use SQLite database. In SQLite database user can insert different html,css,javascript code so that this composer package can be used for different admin themes like AdminLTE, CoreUI.
* Also make automatic Report generator like CRUD (Create, Read, Update, Delete) generator.

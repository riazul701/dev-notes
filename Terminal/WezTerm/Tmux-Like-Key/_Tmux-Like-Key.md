# Setup Instruction

## PATH Details

* WezTerm config file path in Windows-OS: `C:\Users\{{user-name}}\.wezterm.lua`

* WezTerm config base folder path in Windows-OS: `C:\Users\{{user-name}}\.wezterm\`
  * WezTerm considers this folder as base directory of plugins.
  * Inside this folder, create folder `plugins` and git-clone plugin here.

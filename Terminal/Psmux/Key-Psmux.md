# Commands/Usage

# Keybindings from `tmux --help`

* KEY BINDINGS (default prefix: Ctrl+B):

* First press `Ctrl + b` prefix and type command like `:new` (creates new session).

## Sessions

* `prefix + s` : Session chooser

* `prefix + $` : Rename session

* `prefix + d` : Detach from session

## Windows

* `prefix + c` : Create new window

* `prefix + ,` : Rename current window

* `prefix + &` : Kill current window

* `prefix + w` : Window/pane chooser

* `prefix + n` : Next window

* `prefix + p` : Previous window

* `prefix + 0-9` : Select window by number

* `prefix + l` : Last window

* `prefix + '` : Select window by index

## Panes 

* `prefix + i` : Display pane info

* `prefix + ;` : Last pane

* `prefix + %` : Split pane left/right

* `prefix + "` : Split pane top/bottom

* `prefix + {` : Swap pane up

* `prefix + }` : Swap pane down

* `prefix + Arrow` : Navigate between panes

* `prefix + Space` : Next layout

* `prefix + o` : Switch to next pane

* `prefix + q` : Display pane numbers

* `prefix + z` : Toggle pane zoom

* `prefix + !` : Break pane to new window

* `prefix + C-Arrow` : Resize pane by 1

* `prefix + M-Arrow` : Resize pane by 5

* `prefix + x` : Kill current pane

## Mics

* `prefix + [` : Enter copy/scroll mode

* `prefix + ]` : Paste from buffer

* `prefix + =` : Buffer chooser

* `prefix + :` : Enter command mode

* `prefix + ?` : List keybindings

* `prefix + t` : Clock mode

* `prefix + M-1..5` : Preset layouts

## COPY MODE KEYS (prefix + [):

* `↑/k` : Scroll up

* `↓/j` : Scroll down

* `PgUp/b` : Page up

* `PgDn/f` : Page down

* `g` : Top of scrollback

* `G` : Bottom

* `←/h` : Cursor left

* `→/l` : Cursor right

* `w/W` : Next word

* `b/B` : Previous word

* `0` : Start of line

* `$` : End of line

* `^` : First non-blank

* `H/M/L` : Top/Mid/Bot

* `f/F` : Find char fwd/bwd

* `t/T` : Till char fwd/bwd

* `%` : Matching bracket

* `{/}` : Prev/next paragraph

* `/` : Search forward

* `?` : Search backward

* `n` : Next match

* `N` : Previous match

* `v` : Rectangle toggle

* `V` : Line selection

* `Space` : Begin selection

* `y/Enter` : Yank (copy)

* `D` : Copy to end of line

* `"a-z` : Named registers

* `o` : Swap cursor/anchor

* `1-9` : Numeric prefix

* `q/Esc` : Exit copy mode

# Widget-Yasb.md

## Notes

* LaunchpadWidget supports auto-completion.

* In NotesWidget, `Enter` creates new note. Press `Shift + Enter`, to create new line.

* In QuickLaunchWidget, press `Space` after entering prefix, to acivate different providers like Apps, Bookmarks, Clipboard History etc.
  * Turn on Clipboard History: Goto, Windows Menu -> Settings -> System -> Clipboard, then enable "Clipboard history"

* In WallpapersWidget, on gallery view press `Enter` to set wallpaper on windows desktop.

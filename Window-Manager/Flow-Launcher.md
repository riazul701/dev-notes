# Configuration

**`Alt+d` : Open Flow-Launcher**

* Open Flow-Launcher Search Bar -> Right Click -> Settings -> Hotkeys -> Open Flow Launcher (Enter shortcut to show/hide Flow Launcher) -> Set `Alt+d` (like dmenu in i3 window manager)

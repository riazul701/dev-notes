# Map SMB Share

## Windows-OS

This is best if you want smoother browsing or to use playlists.

1. Open `File Explorer`

2. Right-click `This PC → Map network drive`

3. Choose a drive letter (e.g., `Z:`)

4. Folder:

`\\\SERVER\_IP\ShareName` (Find this address in shared folder -> Right Click -> Properties -> Sharing)

5. Enter username/password if needed

6. Click `Finish`

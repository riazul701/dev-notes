# Important

* Do Not Place/Transport Ethernet Cable (Cat6, Cat7, Cat8) With 220 Volt Electricity Cable. If There Is Any Leakage, 220 Volt Electricity Will Damage/Destroy Electronic Devices Like Computer, Television, WiFi Router.

* Use Separate Two Plastic Pipes To Transport 220 Volt Electricity Cable And Ethernet Cable (Cat6, Cat7, Cat8).

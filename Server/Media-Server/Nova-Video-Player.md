# Nova-Video-Player.md

## Delete Caution

* Nova-Video-Player has file `delete` option.

* Create a windows-os user. Share media directory with read-only permission for that user. On nova video player login with that user via SMB. 

## Notes

* Nova video player shows interface like Kodi.

* Nova video player fetches movies and TV shows metadata from online.

* It does not use Kodi's offline metadata.

* It downloads subtitle from [opensubtitles.org](https://www.opensubtitles.org/en/) during playback.

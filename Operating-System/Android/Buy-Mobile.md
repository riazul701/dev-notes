# Buy-Mobile.md

**MUST BUY "USB TYPE-C" CHARGING PORT INCLUDED ANDROID MOBILE**

**ANDROID MOBILE MUST HAVE "OTG" SUPPORT**

## Wired Keyboard & Mouse Support

* On showroom display mobile, check wired keyboard and mouse supports or not.

* Use "USB Type-C" USB HUB to connect wired keyboard and mouse.

* Do not use backlit keyboard and mouse, because they will drain mobile battery quickly.

## Application Support

**On Showroom Display Mobile, Check These Softwares Installed Properly Or Not**

* Put android software's `.apk` file in google drive and share it as public. Create a github public project and write google drive public shared links on markdown file. Visit this markdown file from showroom display mobile.

* TERMUX (MOST IMPORTANT) (Does not support all mobiles)

# Commands/Usage

# General

## Common

* `:checkhealth` : Check plugins dependencies are installed or not.

* `:checkhealth mason` : Shows LSP, DAP, Linter, Formatter Dependencies.

* `:checkhealth snacks` : Check `snacks` plugin status.

## LSP

* `:LspLog` : Shows LSP related logs.

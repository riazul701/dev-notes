# LazyVim.md

## Notes

* Restart `nvim` (LazyVim), to reload configuration.

* If any plugin's functionality does not work, then check plugin is loaded or not inside `lazy.nvim` plugin manager.

* `:checkhealth`, `:checkhealth mason`, `:checkhealth snacks` : Check plugins dependencies are installed or not.

## PATH

* LazyVim starter-kit path: `C:\Users\{{user-name}}\AppData\Local\nvim`

* Plugin and language server path: `C:\Users\{{user-name}}\AppData\Local\nvim-data`

* LazyVim-Extra plugin config file path: `C:\Users\{{user-name}}\AppData\Local\nvim\lua\config\options.lua`

# Concept

> ChatGPT Question: 
> 
> Using touch typing method, explain cut, copy, paste, undo finger placement

Here’s how **cut / copy / paste / undo** work using **touch typing (home row method)**.

I’ll assume standard touch typing position:

```text id="t0"
Left hand:  A S D F
Right hand: J K L ;
Thumbs:     Space
```

---

***# ✂️ CUT = Ctrl + X***

```text id="t1"
Left hand:  pinky moves to CTRL (bottom-left)
Right hand:  X = left-hand ring finger (D position → down-left)
```

Finger path:

* Left pinky → Ctrl (hold)
* Left ring finger → X

---

***# 📋 COPY = Ctrl + C***

```text id="t2"
Left hand:  pinky holds CTRL
Right hand:  C = left-hand index finger (F position → down-left)
```

Finger path:

* Left pinky → Ctrl (hold)
* Left index finger → C

---

***# 📥 PASTE = Ctrl + V***

```text id="t3"
Left hand:  pinky holds CTRL
Right hand:  V = left-hand index finger (F position → down-left, same area as C but one key right)
```

Finger path:

* Left pinky → Ctrl (hold)
* Left index finger → V

---

***# ↩️ UNDO = Ctrl + Z***

```text id="t4"
Left hand:  pinky holds CTRL
Right hand:  left-hand ring finger (A row → Z)
```

Finger path:

* Left pinky → Ctrl (hold)
* Left ring finger → Z

---

***# 🧠 Touch typing summary***

```text id="t5"
CTRL  = left pinky (always)
Z     = left ring finger
X     = left ring finger area (diagonal reach)
C     = left index finger
V     = left index finger
```

---

***# 🖐️ Key idea (touch typing logic)***

```text id="t6"
All shortcuts stay on LEFT side of keyboard
Only LEFT hand moves for X C V Z
LEFT pinky stays on CTRL
```

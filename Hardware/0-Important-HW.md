# Important.md

## Linux Support

* https://linux-hardware.org/
* Asus desktop motherboard supports linux, without any issue.
* Gigabyte desktop motherboard freezes during linux usage.

## Windows-10 BlueScreen Error

* If Windows-10 "Blue Screen Of Death (BSOD)" occurs repeatedly, then use Windows-X-Lite Optimum-10 (Defender On). If computer shutdown does not work properly on Optimum-10, then first sign out and shutdown from lower right button.

## Linux-OS Freezes

* This error is equivalent of Windows-10 "Blue Screen Of Death (BSOD)".
* This error happens because of mismatched dual ram stick. Remove one ram stick and check problem solves or not.

## DIY 18.5 Inch Desktop Monitor Stand

* HP V194 18.5 inch monitor's dimension
  * Width: 17.5 Inch
  * Height (Monitor Only): 11 Inch
  * Height (With Stand): 13.5 Inch

* For making DIY monitor stand, do not use screw, instead use hinge.

* Use SS (Stainless Steel), not MS (Mild Steel) or plain steel. Because SS steel is stronger and lighter.

* VESA Plate: Monitor VESA mount has two sizes, 75x75cm and 100x100cm. According to VESA size, make a SS plate and make 4 holes for screws. 

* Vertical Stand: Attach verical stand with "VESA Plate" using hinge, so that monitor can be tilted. Hinge must be strong so that, it can hold tilted monitor.
  * Example: [Zeno S510 10-32 Inch LCD/LED/PLASMA TV Wall Mount Bracket](https://www.ryans.com/zeno-s510-10-32-inch-tv-wall-mount-bracket)
* Floor Plate: Attach floor plate with "Vertical Stand" using hinge, so that it can be fully folded at backside of monitor.

* Buy briefcase, by calculating whole monitor setup with DIY stand.

## DIY Foldable Monitor Stand

* Use VESA (100 CM X 100 CM) Mount. Make VESA Mount Using Stainless Steel Plate.

* Choose Appropriate Stainless Steel Square Bar (1.25 Inch X 1.25 Inch) Or (1.5 Inch X 1.5 Inch) Dimension, So That (1 Inch X 1 Inch) Can Easily Be Fit Inside Of It.

* Attach (1.25 Inch X 1.25 Inch) Or (1.5 Inch X 1.5 Inch) Dimension 3 Inch Long Stainless Steel Square Bar To VESA (100 CM X 100 CM) Mount Using Weilding. Make A Screw Hole On Square Bar.

* Make A (8 Inch X 8 Inch) Square Shape Stainless Steel Plate. Attach (1.25 Inch X 1.25 Inch) Or (1.5 Inch X 1.5 Inch) Dimension 2 Inch Long Stainless Steel Square Bar At The Center Of It Using Weilding. Make A Screw Hole On Square Bar.

* Make (1 Inch X 1 Inch) Dimension Stainless Steel Square Bar With 2 Screw Holes On Upper And Lower End. Place It Between VESA Mount Holder And Plate Holder And Attach With Screw Inside Screw Hole. Holder Is (1.25 Inch X 1.25 Inch) Or (1.5 Inch X 1.5 Inch) Dimension Stainless Steel Square Bar.
  * Make Appropriate Length Of This Square Bar So That Monitor's Lower End Stands 4 Inch Above From Ground.

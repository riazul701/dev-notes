# Install

## Fedora

* `sudo dnf install redshift` : Install `redshift` command <sup>{3}</sup>

* `sudo dnf install redshift-gtk` : Install GUI of `redshift` command <sup>{4}</sup>

* `redshift` : Launch `redshift` command

# 23.8Inch-AntiGlare-IPS-Speaker.md

## Monitor Specifications Check

* Size : 23.8 Inch
* Light Reflection / Display Surface : Anti-Glare
* Panel : IPS
* Resolution : Full-HD (1920 x 1080)
* Built-in Speaker : Must Have
* Monitor Input : Both HDMI And VGA
* Contrast Ratio : 1500:1 (Minimum)
* Refresh Rate (Hz) : 144Hz (More, Better)
* Response Time (ms) : 1ms (MPRT), 4ms (Gray to Gray)
* Viewing Angle : 178 degree (Horizontal & Vertical)
* Eye Care : Yes
* Eye Care Features : TUV Certified, MSI EyesErgo with Anti-Flicker Technology, MSI Eye-Q Check.
* Tilt Adjustment : -5 - 20 Degree
* VESA Wall Mount Standard : 100 x 100mm
* Warranty : 3 Year (Box Mandatory While Claiming)

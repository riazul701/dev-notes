# Check-Before-Buying-Monitor.md

## Check Bofore Buying New/Used Monitor

* Install `Twinkle-Tray` or `Monitorian` software and check this software can change brightness and contrast or not.
  * That means, this monitor supports DDC/CI (Display Data Channel / Control Interface) Feature.
  * `HP V194` 18.5 inch monitor supports this feature. Conceptually, any branded monitor released after this `HP V194` monitor, supports DDC/CI feature.

* Play a screen-test video from youtube and check any dead-pixel, stuck-pixel, blue-line, other-line exists or not.
